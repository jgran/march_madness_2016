This analysis makes predictions about the outcomes of the 2016 NCAA men's basketball tournament as part of this kaggle competition:

https://www.kaggle.com/c/march-machine-learning-mania-2016

My strategy is to use differences in season average statistics and tournament seeding as input features. Two separate predictions are made, one using logistic regression and one using a random forest.

To run (about 5 mins on my laptop):
source run.sh

Details on the format of the submission files and the scoring of the competition are given here:
https://www.kaggle.com/c/march-machine-learning-mania-2016/details/evaluation
